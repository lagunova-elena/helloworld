Найденные дефекты

1. Повторное добавление уже добавленного файла в список файлов - программа добавляет отдельной строкой еще один файл с таким же названием

2. Удаление файла из места копирования после добавления в список файлов - после нажатия на кнопку "запись" программа зависает

3. Подмена копируемого файла папкой с таким же расширением - после нажатия на кнопку "запись" программа зависает

4. Копирование файла в место, где уже содержится файл с таким же названием - программа перезаписывает файл, ничего не спрашивая

5. Поле "папка, в которую писать файлы" - при наборе произвольных символов и нажатии на кнопку "запись" программа зависает

6. Копирование файла в место, где уже содержится файл с таким же названием, доступный только для чтения - после нажатия на кнопку "запись" программа зависает

7. При копировании файла в папку, в которой файл находится - программа зависает

8. Для удаления файла необходимо выделить строку, при этом, если контуром мышки выделить все файлы - после нажатия на кнопку "удалить" удаляется только верхний файл из списка

9. Это не дефект, а, скорее, особенность, требующая дополнительных времязатрат: перед записью файлов необходимо активировать чек-боксы напротив них, при этом кливиши быстрого управления (ctrl+A) не работают

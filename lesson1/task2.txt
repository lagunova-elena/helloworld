Тестирование яйца

- взять предмет -> определить форму (овал) -> определить размер (2-8 см в длину, 1-4 см в поперечном разрезе) -> определить цвет (бежевый/белый/пятнистый) -> просветить рентгеном (если внутри видим шар, внутри которого зародыш - это яйцо) -> другой вариант: разбить внешнюю оболочку -> если внутри бесцветная жидкость+желтый мягкий шар - это яйцо;

- взять яйцо -> определить форму/размер/цвет -> покрутить “волчком” на столе -> если останавливается быстро - яйцо сырое -> можно сварить;

- взять яйцо -> определить форму/размер/цвет -> покрутить “волчком” на столе -> если крутится долго - яйцо вареное -> можно съесть.
